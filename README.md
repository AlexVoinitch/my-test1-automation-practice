# Cypress Automation Practice

This repository contains a basic setup for Cypress automation testing.

## Running Cypress

* **Open Cypress in debug mode (with UI):**
    ```bash
    npx cypress open
    ```
* **Run Cypress in headless mode (without UI):**
    ```bash
    npx cypress run
    ```
